# CareerPilot AI

CareerPilot AI is an AI-powered resume analysis platform that helps job seekers evaluate their resumes for specific roles using Google's Gemini AI.

## Features

* PDF Resume Analysis
* DOCX Resume Analysis
* Role-Specific Resume Evaluation
* ATS Score Prediction
* Role Match Score
* Skill Gap Identification
* Interview Readiness Assessment
* Personalized Improvement Suggestions
* AI-Generated Interview Questions

## Tech Stack

* Python
* Flask
* Google Gemini API
* HTML
* CSS
* PDFPlumber
* Python-Docx

## How It Works

1. Upload a PDF or DOCX resume.
2. Enter your target job role.
3. CareerPilot AI analyzes the resume.
4. Receive an AI-generated report containing:

   * ATS Score
   * Role Match Score
   * Strengths
   * Skill Gaps
   * Interview Questions
   * Actionable Recommendations
